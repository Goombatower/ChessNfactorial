# ChessNfactorial

What is this?
This is ChessNfactorial.app , an application specialized for learning and playing chess from the comfort of your home.

Capabilities.
This website has several key features that are vital for the online chess experience. It has an adjustable time control, for both classical games, rapids, and blitz's. Light and Dark mode, for the comfort of your eyes and high intensity area's alike. It has full knowledge of chess rules, and knows how to declare draws and wins, both based on timer, and actual game performance

Plans for the future.
If given further development time and funding, I would be able to add several key features, such as AI opponents of various strength, internet compatibility between different people, and tutorials for openings. These would all be massively beneficial in both teaching you how to play well, as well as entertaining yourself with engaging gameplay against both real and AI opponent.
